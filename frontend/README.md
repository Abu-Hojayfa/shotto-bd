# 🏛️ Government Transparency Platform

A full-stack web application designed to promote transparency,
accountability, and accessibility in government operations. This
platform allows citizens to explore public data, track activities, and
interact with government information through a modern and responsive
interface.

------------------------------------------------------------------------

## 🚀 Tech Stack

### Frontend

-   React.js
-   Tailwind CSS
-   React Router
-   Axios

### Backend

-   Node.js
-   Express.js

### Database

-   MongoDB

------------------------------------------------------------------------

## ✨ Features

-   📊 View and explore public government data
-   🔍 Search and filter records efficiently
-   👥 Clean and user-friendly interface
-   🔐 Secure REST API
-   ⚡ Fast and responsive UI with Tailwind CSS
-   🔄 Scalable full-stack architecture

------------------------------------------------------------------------


## 🛠️ Installation & Setup

### 1. Clone the repository

``` bash
git clone <your-repo-link>
cd <project-folder>
```

### 2. Install dependencies

``` bash
npm install
```

### 3. Setup Environment Variables

Create a `.env` file:

``` env
PORT=5000
MONGO_URI=your_mongodb_connection_string
```

### 4. Run the development server

``` bash
npm run dev
```
